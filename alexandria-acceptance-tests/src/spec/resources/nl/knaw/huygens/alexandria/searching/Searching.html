We can successfully /searches the following body { "query": { "find": "annotation", "where": "who:eq(\"nederlab\")", "sort": "-when", "return": "id,resource.url", "pageSize": 10 } } where the properties of query have the following semantics:
find
the Alexandria concept to look for. (currently annotation is the default as well as the only valid value)
where
used to filter records, you can check the ANTLR v4 grammar file for field names and functions. (caveat: multiple where clauses are treated as a conjunction, meaning ALL clauses have to match)
sort
used to sort the result-set on a field (ref. where). Use + for ascending (default) and - for descending order. (specifying sort is optional, defaulting to -when if left out)
return
used to select which fields (ref. where) from the result-set are returned in the output.
pageSize
how many results from the result-set to include in each page of results. (specifying pageSize is optional, defaulting to 10 results per page if left out)
