Accessing annotations

What happens when I try to GET a non-existent annotation?

  • Given that no annotation exists with ID 3ed4faaa-c0cd-11e4-a84e-83ef41cbdc44
  • When I GET /annotations/3ed4faaa-c0cd-11e4-a84e-83ef41cbdc44
  • Then:
    1. the response status should be 404 Not Found;
    2. the response body is empty.

How do I create an annotation?

  • Given an annotation of type: emotion and value: happy
  • When I POST 
    {
    "annotation": {
        "type": "emotion",
        "value": "happy",
        "createdOn" : "2015-03-23T08:28:36.571Z"
    }
}
    to /annotations
  • Then:
    1. the response status should be 201 Created;
    2. the response contains the Location header https://{host}/annotations/ followed by a well-formed UUID;
    3. the response body looks like: 
      {
    "annotation": {
        "type": "emotion",
        "value": "happy",
        "annotations": [],
        "createdOn": "{date.beforeNow}"
    }
}

How can I create an annotation which is annotated itself by an existing annotation?

  • Given an annotation of type: emotion and value: happy
  • and that another valid annotation exists with id: 7631e9ef-a41e-40fa-a3dc-ba03267ef34a
  • When I POST 
    {
    "annotation": {
        "type": "emotion",
        "value": "happy",
        "annotations" : [ "7631e9ef-a41e-40fa-a3dc-ba03267ef34a" ],
        "createdOn" : "2015-03-23T08:28:36.571Z"
    }
}
    to /annotations
