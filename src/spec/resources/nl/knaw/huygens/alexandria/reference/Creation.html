Resource Endpoint

How can I create a reference resource?

Given a base URI of https://alexandria.huygens.knaw.nl

When I POST the following body to /resources:

{ "resource": { "ref": "http://nederlab.nl/titles/path/to/title/resource" } }

Then the response

  • status should be 201 Created
  • should contain a Location header
    • starting with https://alexandria.huygens.knaw.nl/resources/
    • ending with a valid UUID
  • body is (empty)
