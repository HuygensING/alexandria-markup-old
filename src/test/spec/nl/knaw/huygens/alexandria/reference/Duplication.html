References

A reference in Alexandria refers to an external object that can be annotated.

Before a reference can be given any annotations, it first needs to be created.

Interaction with existing references

Given the following references in the database:

Id
dpo.1234
dpo.8111

Then these are the results of trying to add references given unused or existing Ids:

Use case Id Result
Create a reference with unused Id dpo.3456 201 Created
Create a reference with existing Id dpo.8111 409 Conflict
