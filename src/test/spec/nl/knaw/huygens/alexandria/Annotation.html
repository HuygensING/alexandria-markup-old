Annotation requirements

Irrespective of what is in the database:

  • Adding annotation some_annotation to a reference with a null id returns 400 Bad Request.
  • Creating annotation some_annotation to a reference with an empty ("") id returns 400 Bad Request.

  • Adding the annotation "some annotation" to a reference with iddpo.8111 returns 201 Created.
